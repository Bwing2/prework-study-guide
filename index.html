HTML

  • The head element contains information about the webpage.
  • Head and meta data are not visable content on website.
  • The body element represents the visible content shown to the user.

CSS

  • 3 ways to style webpages using CSS, inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS applies to a specific HTML tag, using a style attribute for a particular page element only.
  • Internal style sheet embeds entire stylesheet directly in the HTML file using style element in the head element.
  • External style sheet is most common as it allows devs to keep their CSS rules in a separate file, making design changes easier.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • All git statements must be proceded with git command.
  • code .: opens VS Code.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: Add command adds modifications in current working branch area. flag indicates we want all changes (-A).
  • git commit -m "add starter code": commits with a message describing what the commit contains (-m).
  • git pull origin main: recieves branch's modification in local environment (pull), and source of pull will be in GitHub repo (origin), in the "main" branch.
  • git push origin branch-name: pushes changes locally to remote GitHub branch, and creates a new remote branch/adds changes to existing branch (push).

JavaScript

  • Variable is a named container that allows us to store data in our code.
  • “var” keyword is used to declare a variable, after var give variable a unique name.
  • = is an assignment operator that assigns specific data we want stored in our variable.
  • Connect script.js by adding code under footer element and above closing element for body
  • console.log method can be seen in chrome developer tools when inspecting a page, and clicking the Console tab.
  • === is the strict equality operator, and it checks to see if 2 values are equal, and returns a boolean result of true or false.
